1. Descripción del proyecto: 
El proyecto consiste en el diseño de un sitio web para un club deportivo de la localidad de Bakio. El proyecto se enmarca en el programa educativo del Bootcamp de Desarrollo Fullstack de la institución Peñascal - F5. El proyecto consiste en el diseño de cinco (05) páginas: 1. Inicio 2. Sobre Nosotros, 3. Servicios, 4. Galeria de imágenes y contacto.
